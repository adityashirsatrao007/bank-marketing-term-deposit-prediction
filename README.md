# Aegis Term Deposit Predictor: Serverless ML Pipeline & Glassmorphic Dashboard

[![Aegis ML CI Pipeline](https://github.com/adityashirsatrao007/bank-marketing-term-deposit-prediction/actions/workflows/ci.yml/badge.svg)](https://github.com/adityashirsatrao007/bank-marketing-term-deposit-prediction/actions)
[![License: MIT](https://img.shields.io/badge/License-MIT-blue.svg)](https://opensource.org/licenses/MIT)
[![Python Version](https://img.shields.io/badge/Python-3.8%2B-green.svg)](https://python.org)

**Aegis Predict** is a state-of-the-art, end-to-end Machine Learning classification and interactive simulation platform designed to predict customer conversion for bank term deposit campaigns (using the UCI Bank Marketing Dataset). It compiles trained model coefficients (Logistic Regression) and recursive regression trees (Gradient Boosting) into a high-performance, serverless client-side engine running directly in vanilla Javascript with micro-second prediction latency.

---

## 🖥️ Interactive Glassmorphic Dashboard
The system comes packaged with a premium, responsive **visual analytics dashboard** featuring a fully client-side real-time simulator.

- **Serverless Predictor**: Move sliders and toggle dropdown values—predictions update instantly in-browser using the serialized model weights (Logistic Regression & Gradient Boosting are computed live in pure JS!).
- **Dynamic Charting**: Inspect real-time performance curves (ROC Curve, Confusion Matrix, and Feature Importance) generated by the Python pipeline.
- **Micro-Animations**: Hover states, smooth gradients, and glassmorphic UI cards using the latest Web Design tokens.

---

## 📊 Model Architecture & Evaluation Results

The models are trained using a stratified 80/20 train/test split. Gradient Boosting outperforms as the superior classifier with an outstanding **92.90% ROC-AUC**.

| Evaluation Metric | Logistic Regression | Gradient Boosting (Aegis Ensemble) |
| :--- | :---: | :---: |
| **Accuracy** | 90.05% | **90.76%** |
| **Precision** | 64.11% | **65.90%** |
| **Recall** | 33.93% | **43.48%** |
| **F1-Score** | 44.38% | **52.39%** |
| **ROC-AUC** | 90.61% | **92.90%** |

## 🛠️ Repository & File Map

- **`.github/workflows/ci.yml`**: GitHub Actions automated workflow running pipeline dry-runs and unit tests on check-ins.
- **`src/`**: Modularized ML pipeline code.
  - **`config.py`**: Centralized configurations and file paths.
  - **`data_ingestion.py`**: Handles dataset downloading, zip extraction, and schema checks.
  - **`preprocessor.py`**: Automated numeric scaling and dummy categorical encoding.
  - **`models.py`**: Model building, hyperparameters configuration, and pickle serializations.
  - **`evaluation.py`**: Matplotlib generation of ROC curves, confusion matrices, and feature importances.
  - **`export_model.py`**: Compiles scales and ensemble trees into `model_bundle.json`.
- **`tests/test_pipeline.py`**: Automated unittest suite verifying splits, training shapes, and serialization logic.
- **`web/`**: Code for the glassmorphic interface and JavaScript inference engine.
- **`run.py`**: The master pipeline runner and dashboard web server.

## 🚀 Quick Start & Installation

### 1. Clone the repository
```bash
git clone https://github.com/adityashirsatrao007/bank-marketing-term-deposit-prediction.git
cd bank-marketing-term-deposit-prediction
```

### 2. Install dependencies
```bash
pip install -r requirements.txt
```

### 3. Run Pipeline & Start Interactive Dashboard
Simply execute the master orchestrator script:
```bash
python run.py
```
This single command will:
1. Fetch and preprocess the dataset.
2. Train the models and generate performance visual plots.
3. Export serialized weights to the assets directory.
4. Launch a local web server at **http://localhost:8080** and open the glassmorphic dashboard automatically in your default browser!

### 🧪 Running Unit Tests
To execute the automated pipeline checks:
```bash
python -m unittest tests/test_pipeline.py
```
